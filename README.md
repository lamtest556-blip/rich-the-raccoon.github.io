## ✏️ 如何更新内容

### GitHub 网页编辑（最简单）

1. 访问仓库：https://github.com/lamtest556-blip/rich-the-raccoon.github.io
2. 点击要编辑的文件
3. 点击右上角的铅笔图标 ✏️
4. 修改内容
5. 填写提交信息并点击 "Commit changes"
6. 等待 1-2 分钟，GitHub Pages 会自动更新

### 本地编辑后推送

```bash
git clone https://github.com/lamtest556-blip/rich-the-raccoon.github.io.git
cd rich-the-raccoon.github.io
# 编辑文件
git add .
git commit -m "更新博客内容"
git push origin main
```
